Acerca del Proyecto

Este proyecto tiene como objetivo mostrar resúmenes estadísticos relacionados con la precipitación anual. Los usuarios pueden interactuar con gráficos dinámicos para observar las tendencias de lluvia a lo largo del tiempo y descargar los datos completos en formato CSV para un análisis más detallado.
